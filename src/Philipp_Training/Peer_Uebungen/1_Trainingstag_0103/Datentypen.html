Primitive Datentypen in Java




Einleitung

Es gibt 8 primitive Datentypen in Java. String ist keiner davon.
Damit können Wahrheitswerte, einzelne Zeichen, ganze sowie Gleitkommazahlen
gespeichert werden.


Zusammenfassung der Datentypen

Datentyp Größe Wertebereich
boolean
1 Bit true oder false
char
16 Bit / 2 Byte 1 Zeichen von Unicode UTF-16 (65.536 Zeichen)
byte
8 Bit / 1 Byte -128 bis 127
short
16 Bit / 2 Byte -32.768 bis 32.767
int
32 Bit / 4 Byte Fies lang
long
64 Bit / 8 Byte Fies lang und noch länger
float
32 Bit / 4 Byte Fies lang mit Komma
double
64 Bit / 8 Byte Fies lang und noch länger mit Komma
